Instruções para o Script edxpgtoolbuilder

1. Baixar o Arquivo

Use o comando wget para baixar o arquivo edxpgtoolbuilder.tar.gz do repositório GitHub:

wget https://github.com/edxlinux993/edxpgtool/raw/main/edxpgtoolbuilder.tar.gz

2. Descompactar o Arquivo

Após o download, descompacte o arquivo com o comando tar:

tar -vzxf edxpgtoolbuilder.tar.gz

3. Entrar no Diretório

Entre no diretório descompactado:

cd edxpgtoolbuilder

4. Executar o Script

Execute o script edxpgtoolbuilder.sh:

./edxpgtoolbuilder.sh

Mensagem Final

Após a execução, o script exibirá a mensagem:

edxlinux e bigOs agradecem
